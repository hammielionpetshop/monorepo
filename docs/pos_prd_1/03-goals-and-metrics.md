# 3. GOALS & SUCCESS METRICS

## 🎯 Business Goals

| Goal | Metric | Target |
|------|--------|--------|
| Kecepatan transaksi | Avg time per transaction | < 30 detik |
| Akurasi stock | Stock variance per bulan | < 1% |
| Uptime POS | Availability | > 99.5% |
| Offline capability | Waktu offline max | Unlimited (sampai internet kembali) |
| Data loss | Kehilangan transaksi offline | 0% |

## 👤 User Goals

| User | Goal |
|------|------|
| Kasir | POS cepat, mudah, tidak hang saat offline |
| Manager | Real-time monitoring, approval workflow jelas |
| Owner | Laporan akurat, kontrol penuh multi-cabang |
| Gudang | Terima barang dengan cepat meski invoice belum datang |

## 📊 Success KPIs

- ✅ Semua 20 toko bisa operasi simultan tanpa conflict
- ✅ Transaksi offline tersync 100% saat online kembali
- ✅ Settlement zero-tolerance tercapai (semua selisih tercatat)
- ✅ FIFO costing akurat sampai ke level UOM
- ✅ Auto-break multi-UOM berfungsi tanpa error
