# 5.1 DASHBOARD & KPI

## 5.1.1 Dashboard Overview

**Target User:** Owner, Finance, Manager Backoffice

**Layout:**
- **Top Bar**: Company name, user profile, notification bell, quick actions
- **Main Content**: Cards + Charts + Tables
- **Sidebar**: Navigation menu (collapsible)

## 5.1.2 Dashboard Scope

**Dua Mode:**
1. **Per Cabang**: Pilih 1 cabang → lihat detail cabang tersebut
2. **Konsolidasi (Agregat)**: Lihat summary SEMUA cabang

**Toggle Switch:**
```
┌────────────────────────────────────┐
│ [Per Cabang ▼] | [Agregat Semua]  │
└────────────────────────────────────┘
```

```
[Hari Ini] [Minggu Ini] [Bulan Ini] [Custom Range]
```

## 5.1.4 Charts

**Chart 1: Sales Trend (Line Chart)**
- X-axis: Tanggal (last 7/30 days)
- Y-axis: Sales (Rp)
- Multi-line: 1 line per cabang (bisa toggle on/off)

**Chart 2: Sales by Category (Pie Chart)**
- Categories: Makanan, Obat, Aksesoris, dll
- Show percentage & nominal

**Chart 3: Top 10 Products (Bar Chart)**
- X-axis: Product name
- Y-axis: Qty sold
- Sortable (by qty or by revenue)
