# 3.7 BULK OPERATIONS

## 3.7.1 Bulk Edit Products

```
Use case: Edit multiple products sekaligus

FLOW:
1. Filter produk (by kategori, brand, dll)
2. Checklist produk yang mau di-edit
3. Klik "Bulk Edit"
4. Pilih field yang mau diubah:
   ├── Change Category
   ├── Change Brand
   ├── Change Status (Active/Inactive)
   ├── Add Tag
   ├── Remove Tag
   └── Change Supplier
5. Confirm → Bulk updated
```

## 3.7.2 Import/Export Products

### Export

```
Format Excel:
┌──────────────────────────────────────────────────────────────┐
│ SKU | Name | Category | Brand | UOM | Stock | Price_Retail  │
├──────────────────────────────────────────────────────────────┤
│ PKM-001 | Pakan Meow | Makanan>Kucing | Meow | Sak | 50 | 100000 │
└──────────────────────────────────────────────────────────────┘
```

### Import

```
1. Download template Excel
2. Fill data
3. Upload file
4. Sistem validasi:
   ├── SKU exist?
   ├── UOM exist?
   ├── Harga > 0?
   └── Format benar?
5. Preview import (show errors if any)
6. Confirm → Data imported
```
