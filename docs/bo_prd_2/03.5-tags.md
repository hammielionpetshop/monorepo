# 3.5 TAGS & LABELS

## 3.5.1 Tag Types

### Auto-Tag (by System)

Sistem auto-assign tag berdasarkan rules:

| Tag | Rule |
|-----|------|
| "Best Seller" | Total sold > 100 unit/bulan |
| "New Arrival" | Created date < 30 hari |
| "Low Stock" | Stock < minimum stock setting |
| "Near Expired" | Expired date < 30 hari |
| "Slow Moving" | Total sold < 10 unit/bulan |
| "Fast Moving" | Total sold > 50 unit/bulan |

### Manual Tag (by Admin)

Admin bisa assign tag manual:
- "Promo Lebaran"
- "Recommended"
- "Exclusive"
- "Limited Edition"
- (Custom tag apapun)

## 3.5.2 Tag Management

**CRUD Tags:**
```
CREATE Tag:
Input: Tag name, tag color (hex) → Save

Assign Tag to Product:
1. Edit product → Tab "Tags"
2. Multi-select tags (auto + manual)
3. Save

Remove Tag:
Uncheck tag → Save
```

**Tag Display di Product List:**
```
┌────────────────────────────────────────────────┐
│ Pakan Meow Mix                                 │
│ [🔥 Best Seller] [🆕 New Arrival] [💰 Promo]  │
└────────────────────────────────────────────────┘
```
