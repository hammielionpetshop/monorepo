# 3.4 BRANDS MANAGEMENT

## 3.4.1 Brand Structure

```
BRAND (Simple):
├── Brand ID (auto, PK)
├── Brand Name (required, unique)
├── Status (Active/Inactive)
├── Created At
└── Updated At

Notes:
- Tidak ada logo, deskripsi, website (sesuai requirement user)
- Brand hanya untuk grouping/filtering saja
```

## 3.4.2 Brand CRUD

**CREATE:**
```
Input: Brand name → Save
```

**READ:**
```
┌──────────────────────────┐
│ Brand Name    | # Produk │
├──────────────────────────┤
│ Meow          | 150      │
│ ProVet        | 80       │
│ PetLove       | 60       │
└──────────────────────────┘
```

**UPDATE:**
```
Edit name → Save
```

**DELETE:**
```
Validasi:
├── Ada produk pakai brand ini? → Block
└── OK → Delete
```
